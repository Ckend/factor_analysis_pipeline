# streamlit_alphalens
基于streamlit的因子分析app

Thanks to https://github.com/hugo2046/streamlit_alphalens

基于原版代码修改了部分逻辑，但目前只支持单个因子的研究，后续希望继续优化。

## 运行

**使用步骤**

0. 请见 data 目录下 factor 和 price 的格式。

1. ```
pip install -r requirements.txt
```

2. ```bash
   # 切换到streamlit_alphalens目录下
   streamlit run app.py
   ```
